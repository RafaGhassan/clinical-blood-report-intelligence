# Clinical Blood Report Intelligence

> AI Agent for automated clinical blood test analysis

## What it does
Analyzes blood test report documents (PDF/text) and answers 
natural language questions about the results — flagging 
abnormal values and explaining their clinical significance.

## Tech Stack
- **Platform**: Langflow
- **Components**: RAG Pipeline, Vector Store, Tool Use
- **LLM**: [nama model yang kamu pakai]

## How to use
1. Import `flows/clinical_blood_report.json` into Langflow
2. Upload a blood test PDF
3. Ask questions in natural language
